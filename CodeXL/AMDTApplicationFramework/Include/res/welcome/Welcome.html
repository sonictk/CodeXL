Create a new project for debugging
Debugging allows you to set breakpoints, watch variables, step through OpenCL kernels and much more.
Automatically selects Debug mode.
Create a new project for profiling
Profile code that runs on the CPU or the GPU. Select a session type to define what data is collected.
Automatically selects Profile mode.
Attach to Process
Select a currently running process and CodeXL will begin collecting CPU profiling data.
The type of data collected is defined by CodeXL's current profile session type.
System-wide Profiling
CodeXL will begin collecting CPU profiling or Power profiling data from all currently running processes.
The type of data collected is defined by CodeXL's current profile session type.
Create a new project for DirectX 12 or Vulkan Frame Analysis
Launch a DirectX 12 or Vulkan based game/graphics application, capture individual frames and analyze their CPU and GPU timeline data
Create a new source file
Create a source file, and build it for multiple hardware targets, to analyze the generated ISA, IL and performance statistics.
The supported platforms are: OpenGL, Vulkan, DirectX and OpenCL.
Add an existing source file
Add an existing source file, and build it for multiple hardware targets, to analyze the generated ISA, IL and performance statistics.
The supported platforms are: OpenGL, Vulkan, DirectX and OpenCL.
